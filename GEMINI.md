# Project Overview

This is a simple web application that allows users to generate a direct download link for a Visual Studio Code extension from its Marketplace URL.

The application is built with:
- HTML
- CSS (with Bootstrap)
- JavaScript

# Running the Project

This is a static web project. To run it, simply open the `index.html` file in a web browser. No special build steps are required.

# Development Conventions

The project uses standard HTML, CSS, and JavaScript. The styling is based on the Bootstrap framework. The JavaScript code is self-contained in `script.js` and uses browser APIs. There are no automated tests.
